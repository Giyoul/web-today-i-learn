### 4. 최종적으로 crew 테이블 생성:
```sql
CREATE TABLE crew (
crew_id INT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
nickname VARCHAR(50) NOT NULL
);
```

### 5. attendance 테이블에서 크루 정보를 추출해서 crew 테이블에 삽입하기:
```sql

SELECT DISTINCT nickname 
FROM attendance;
```

## 문제 12: 가장 늦게 하교한 크루 찾기
```sql
SELECT 
    c.nickname, 
    a.end_time
FROM attendance a
JOIN crew c ON a.crew_id = c.crew_id
WHERE a.attendance_date = '2025-03-05'
ORDER BY a.end_time DESC
LIMIT 1;
```

## 문제 13: 크루별로 '기록된' 날짜 수 조회
```sql
SELECT
    c.nickname,
    COUNT(a.attendance_date) AS total_days
FROM attendance a
         JOIN crew c ON a.crew_id = c.crew_id
GROUP BY c.nickname
ORDER BY total_days DESC;
```

## 문제 14: 크루별로 등교 기록이 있는(start_time IS NOT NULL) 날짜 수 조회
```sql
SELECT 
    c.nickname, 
    COUNT(a.attendance_date) AS check_in_days
FROM attendance a
JOIN crew c ON a.crew_id = c.crew_id
WHERE a.start_time IS NOT NULL
GROUP BY c.nickname
ORDER BY check_in_days DESC;
```

## 문제 15: 날짜별로 등교한 크루 수 조회
```sql
SELECT 
    attendance_date, 
    COUNT(crew_id) AS attendee_count
FROM attendance
WHERE start_time IS NOT NULL
GROUP BY attendance_date
ORDER BY attendance_date ASC;
```

## 문제 16: 크루별 가장 빠른 등교 시각(MIN)과 가장 늦은 등교 시각(MAX)
```sql
SELECT 
    c.nickname, 
    MIN(a.start_time) AS earliest_check_in,
    MAX(a.start_time) AS latest_check_in
FROM attendance a
JOIN crew c ON a.crew_id = c.crew_id
WHERE a.start_time IS NOT NULL
GROUP BY c.nickname
ORDER BY earliest_check_in ASC;
```
